Installation & Running

Follow these steps in your terminal to get the application running locally.

  1. Clone the Repository
    Download the source code to your local machine. 
    git clone https://github.com/nozomi-75/soko.git
  2. Navigate into the Directory
    Move into the newly created project folder. 
    cd soko
  3. Run the Application
    Use the Maven Wrapper included in the project. This will download dependencies and start the server.
    For macOS / Linux: 
    ./mvnw spring-boot:run
    For Windows: 
    mvnw.cmd spring-boot:run
  4. Access the Application
    Once the server starts, open your web browser and go to:
    http://localhost:9090
