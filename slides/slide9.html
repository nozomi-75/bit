Project Dependencies

This project is built using a number of excellent open-source libraries. A big thank you to the developers who make these tools possible!

  • Spring Boot:
    A powerful framework for creating production-grade, stand-alone Spring-based applications.
  • Spring Data JPA:
    Provides easy-to-use data access and persistence with a variety of data stores.
  • H2 Database:
    An in-memory, open-source relational database that is ideal for development and testing.
  • Lombok:
    A library that helps reduce boilerplate code for data objects.
  • Apache Commons CSV:
    Provides an API for reading and writing CSV files.
  • Bootstrap & Bootstrap Icons:
    The toolkit for building the responsive user interface and for a comprehensive icon library.
  • Mermaid:
    A JavaScript tool for generating diagrams and flowcharts from text-based descriptions.

About Sōko

Sōko (倉庫), which means "warehouse" in Japanese, is a simple, responsive web application designed for managing a product inventory.

The project serves as a demonstration of a full-stack application built with Spring Boot, H2 Database, JavaScript, and Bootstrap 5. It showcases key concepts like RESTful API services and a mobile-first user interface.
