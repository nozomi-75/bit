Core Features

  • View Products

    Retrieve and display a complete list of all products from the backend with a single API call.

  • Add Products

    Create new product entries via a dynamic form, which sends a POST request to the API.

  • Update Products

    Modify existing product details by sending PUT requests to the API, all while ensuring data consistency.

  • Delete Products

    Remove products from the inventory by sending a DELETE request to the server, handled with a confirmation modal for user experience.

Enhanced Functionality

  • Dynamic Search

    Query the product list as you type and filter results in real-time.

  • Sorting & Filtering

    Reorder products and filter by category with seamless integration.

  • Mobile-First Design

    Responsive interface, so you have a great experience on any device.
