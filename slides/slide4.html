API Endpoint Breakdown

  • GET /api/products

    Retrieves a list of all products. Supports optional query parameters for searching, sorting, and filtering.

  • GET /api/products/{id}

    Fetches a single product by its unique ID.

  • POST /api/products

    Adds a new product to the inventory.

  • PUT /api/products/{id}

    Updates an existing product's details.

  • DELETE /api/products/{id}

    Deletes a product by its unique ID.

RESTful API

The backend is a robust RESTful API built with Spring Boot. It provides a clean set of endpoints that allow the client to perform all necessary CRUD operations on the product data.

Try It Yourself!

Here are some example curl commands to demonstrate how to interact with the API. Remember to run the server first. 

curl -X GET http://localhost:9090/api/products
curl -X POST -H "Content-Type: application/json" -d '{"itemName": "New Item", "itemPrice": 12.99, "itemCategory": "Gadgets"}' http://localhost:9090/api/products
curl -X PUT -H "Content-Type: application/json" -d '{"itemName": "Updated Item", "itemPrice": 15.50}' http://localhost:9090/api/products/1
