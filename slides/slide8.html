The Build Process

  • The Maven Wrapper

    The mvnw command is a wrapper script that downloads and uses the specified version of Maven, so you don't need to install it manually.

  • The pom.xml File

    This file is the core build configuration that declares the project's dependencies, the plugins needed, and other essential project metadata.

  • Generating the JAR

    Maven reads the pom.xml, downloads the necessary dependencies, and uses the Spring Boot Maven Plugin to create a single, executable JAR file.

    mvnw spring-boot:run
