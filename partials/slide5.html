Additional Features

  • CSV Export

    A dedicated backend service leverages the Apache Commons CSV library and queries the database. The product data is streamed back to the user as a downloadable file.

  • Print-Friendly Output

    The application applies a custom stylesheet to hide non-essential UI elements like buttons and modals. This provides a clean, professional, print-friendly look of the product table.

  • Currency Conversion

    This feature is a client-side implementation using vanilla JavaScript. The application script fetches real-time exchange rates from a third-party API and instantly updates all product prices on the screen without a page reload.

Key Files & Libraries

  • CsvExportService.java

    This backend service handles the logic for generating and writing the CSV file. It's the core component for the export functionality.

  • print.css

    A separate stylesheet dedicated to defining print styles, ensuring a clean and readable hardcopy of the inventory list.

  • main.js

    Contains the JavaScript functions for fetching and applying currency exchange rates, as well as the logic for the light/dark theme toggle.

  • Apache Commons CSV

    A powerful Java library used on the backend to simplify the process of reading and writing various CSV formats.

  • ExchangeRate-API

    A free and reliable third-party service that the frontend calls to get up-to-the-minute currency exchange rates.
